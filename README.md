## 📌 Project Overview
End-to-end Data Analytics project analyzing **30,000 e-commerce 
transactions** using PostgreSQL, Python, and Power BI.

**Dataset:** E-Commerce Orders Dataset 2026 (Kaggle)
**Records:** 30,000 | **Features:** 41 | **Dashboard Pages:** 5

---

## 🎯 Business Problem
- Return rate of **10.11%** with no root cause visibility
- No insight into which categories/countries drive revenue
- Coupon strategy impact on profit was unknown

---

## 🛠️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| PostgreSQL + pgAdmin 4 | Database setup & SQL queries |
| Python (Pandas, Seaborn) | EDA — 17 steps, 21 charts |
| Power BI Desktop | 5-page interactive dashboard |

---

## 📊 Dashboard Pages
1. **Executive Summary** — Revenue, Orders, KPIs
2. **Sales & Product Performance** — Category, Season trends
3. **Customer Analytics** — Age groups, CLV, Segments
4. **Shipping, Delivery & Returns** — Operations analysis
5. **Profitability & Marketing** — Margins, Channels, Discounts

---

## 💡 Key Insights
- ✅ Electronics = **55% of total revenue** (6.3M)
- ✅ Fashion has **highest return rate**
- ✅ Social Media = **#1 revenue channel** (~2M)
- ✅ Coupons **reduce profit** without increasing order value
- ✅ Age group **35-44** places most High Value Orders
